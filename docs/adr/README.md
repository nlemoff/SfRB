# Architecture Decision Records

This directory captures significant architectural decisions using lightweight ADRs.

Each record is immutable once accepted. To change a decision, add a new ADR that supersedes the old one and update the old record's status.

## Format

```
# ADR-NNNN: Title

- Status: Proposed | Accepted | Superseded by ADR-XXXX
- Date: YYYY-MM-DD

## Context
## Decision
## Consequences
```
